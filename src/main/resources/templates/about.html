About Contractor Application

 

This is demo application which is being developed for learning purpose. It is small web application which will helps easily manage orders and contractors. Some features are not implemented yet, since the app is in early development phase but finally this app will have following functionality:

  • Adding, editing and deleting contractors and information about contractors including address, total amount of orders, contractors rating etc.
  • adding, editing and deleting orders
  • Multi stages contractor selection process: selection card with recommended contractor will be prepared by user and after manager approval order will be automatically added to the system (this feature is not implemented yet)
  • Multi-currency support. Exchange rates downloaded from the NBP Web API (this feature is not implemented yet)
  • Generating pdf with orders (is to be fixed since UTF-8 is not supported
  • Statistics, charts and analysis of historical data (i.a. amount of orders for contractors in time, percentage of orders completed in time)
  • Searching engine (still not implemented).
  • Exception handling (is still not implemented in 100% - wrong data sent by form can cause illegal exception).

Used technologies: Java 11, Spring Boot, Maven, H2 database, Flyway, Hibernate, Thymeleaf, Keycloack, SpringSecurity, Lombok, JavaScript

Information about server:

  • Server: Rapspberry Pi 4 B
  • System: Linux Raspbian 11.0.8
  • Web App Container: Apache Tomcat 8.5.38 (Debian)
